What is PyTS3?
--------------

It's a small package that contains a Python 3.2+ API for

   * **TS3 Server Query Events**,
   * the **TS3 Server Queries API**
   * the **TS3 File Transfer Interface**,
   * the **TS3 Client Query API**.
